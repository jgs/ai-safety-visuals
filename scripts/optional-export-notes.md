# Optional Export Notes

This repository is SVG-first. PNG exports are optional and should be generated from the source SVGs.

Recommended export sizes:

- LinkedIn single image: 1600 x 900 px
- LinkedIn carousel slide: 1600 x 1600 px
- X image: 1600 x 900 px
- Presentation slide: 1920 x 1080 px

Quality checklist before exporting:

- Text remains readable at mobile preview size.
- Source IDs are visible or included in the caption.
- The graphic does not imply measured data unless it uses real data.
- The image title works without extra context.
- Alt text is written before posting.
